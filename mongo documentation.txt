MongoDB

Installing:

cd C:
cd mongodb/bin
mongod --directoryperdb --dbpath C:\mongodb\data\db --logpath C:\mongodb\mongo.log --logappend --rest --install

--------------------------------------------------------------------------------------------------------------
Start Mongodb:

net start MongoDB


------------------------------------------------------------------------------------------------------------
To enter into shell:
>mongo

To check the number of databases:
>show dbs

Create new DB:
>use databasename

To check which current DB ur in:
>db
